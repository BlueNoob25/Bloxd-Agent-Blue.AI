# How I test the AI
This file explains how I test the AI after each update for each mode.

## Chatbot
I would ask Bloxd Agent Blue about a piece of information about Bloxd.io regarding the knowledge I had intended to train the AI with. For example, if I told the AI that players like shorter variable names, then I would ask the AI to generate a code that would fit the preferences of the general community.
## Code Helper
I usually use this prompt:
`"Make a code where if a player says hi they will get a diamond and they will do a waving animation."`
This helps me test:
- The AI's ability to put prompts into actual code
- The AI's ability to make in-game animations
- The AI's ability to give players items
- The AI's ability to use the correct callback in a scenario
- The AI's ability to work with in-game chat.
- Reprompting (usually by changing the diamond to gold bars)
- Memory updating (usually by asking to change the animation)
- The Right-click to explain feature (usually by asking to remove all the comments in the generated code)
## Post Analyst
This might seem a bit short but I just ask the AI to analyze @Crownix' (the owner of Bloxdhub's) latest post or whatever post appears to me during a short time of doom-scrolling on the Bloxd subreddit or Bloxdhub.
## Mod Constructor
I ask it the same thing as I do in Code Helper mode, but with a full package of course.
